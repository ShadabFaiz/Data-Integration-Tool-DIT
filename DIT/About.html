This OPEN SOURCE Project is being developed to convert the MySQL query into MongoDB query
so that one can save the time that might have been wasted on learning new syntax.Currently,
the development is still on early stage, kindly do not totally depend on it.As of right now,it
can convert only basic queries like create of databases,tables.Further funtionality to process
complex queries will be added soon.This project is being developed by GOOBAR.
